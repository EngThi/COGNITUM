# Flashcards: Idea: Adotar MCP
Origin Note Event ID: 56

### Q1: O que significa a sigla MCP no contexto de sistemas de IA?
**A**: MCP significa Model Context Protocol, um protocolo para integração de ferramentas e contexto em sistemas de IA.

### Q2: Qual é o propósito principal do MCP (Model Context Protocol)?
**A**: Integrar ferramentas e contexto ao sistema, permitindo que modelos de IA acessem dados e funcionalidades externas de forma padronizada.

### Q3: Por que adotar o MCP em vez de integrações ad-hoc?
**A**: O MCP oferece uma interface padronizada, reduzindo a complexidade de integrar múltiplas ferramentas e contextos de forma consistente.
