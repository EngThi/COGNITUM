# Flashcards: Idea: Integração com MCP e Composio
Origin Note Event ID: 39

### Q1: O que é MCP no contexto de integração com IA?
**A**: MCP (Model Context Protocol) é um protocolo aberto que permite a conexão de modelos de IA com ferramentas externas, expandindo suas capacidades de execução de tarefas.

### Q2: Qual a diferença entre MCP e Composio?
**A**: MCP é um protocolo aberto para conectar IA a ferramentas, enquanto Composio é uma plataforma específica que provê integrações prontas e ferramentas para agentes de IA.

### Q3: Qual o objetivo da integração de IA com MCP e Composio?
**A**: Expandir as capacidades de execução de tarefas da IA, permitindo que ela utilize ferramentas externas do ecossistema de forma mais ampla e eficiente.

### Q4: Como a Composio pode ser utilizada por uma IA?
**A**: A Composio oferece ferramentas e integrações prontas que a IA pode utilizar para executar ações em diversos serviços e plataformas sem precisar construir as integrações do zero.
