# Flashcards: MCP e Composio Tools
Origin Note Event ID: 37

### Q1: O que é MCP (Model Context Protocol)?
**A**: MCP (Model Context Protocol) é um protocolo aberto desenvolvido pela Anthropic para padronizar como modelos de IA se conectam a fontes de dados e ferramentas externas, permitindo que diferentes LLMs acessem contexto de forma uniforme.

### Q2: Qual é a função das ferramentas (tools) no Composio?
**A**: As ferramentas do Composio são integrações prontas que permitem que agentes de IA interajam com diversos serviços e APIs (como GitHub, Slack, Notion, etc.) sem necessidade de implementação manual de cada integração.

### Q3: Como MCP e Composio Tools se relacionam?
**A**: O Composio fornece um conjunto de ferramentas prontas que podem ser expostas através do protocolo MCP, permitindo que qualquer modelo compatível com MCP acesse centenas de integrações do Composio de forma padronizada.

### Q4: Quem desenvolveu o Model Context Protocol (MCP)?
**A**: O MCP foi desenvolvido pela Anthropic.

### Q5: Qual problema o MCP resolve no ecossistema de IA?
**A**: O MCP resolve o problema da fragmentação de integrações, criando um padrão único para que modelos de IA de diferentes fornecedores se conectem a fontes de dados e ferramentas externas de forma consistente.

### Q6: O que significa dizer que o MCP é um protocolo 'aberto'?
**A**: Significa que qualquer desenvolvedor ou empresa pode implementar o MCP, não sendo restrito a um único fornecedor de modelos de IA, o que promove interoperabilidade entre diferentes sistemas.
