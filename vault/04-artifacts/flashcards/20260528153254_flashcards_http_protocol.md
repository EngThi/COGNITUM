# Flashcards: HTTP Protocol
Origin Note Event ID: 7

### Q1: Qual é uma característica fundamental do protocolo HTTP em relação ao estado de sessão?
**A**: O protocolo HTTP é stateless (sem estado).

### Q2: Em qual porta padrão o protocolo HTTP opera?
**A**: O HTTP roda sobre a porta 80.

### Q3: Cite alguns verbos HTTP comuns utilizados para operações em recursos.
**A**: Verbos HTTP comuns incluem GET, POST, PUT, DELETE.
