# COGNITUM SYSTEM

COGNITUM is a hybrid cognitive automation runtime.

Core:
- Ubuntu 25.10 VPS
- Python 3.13
- FastAPI
- SQLite append-only event store
- Markdown vault
- async Python workers
- systemd services/timers

Cloud edge:
- Telegram
- Gemini
- Perplexity
- Pipedream
- Composio
- Notion
- Google Tasks
- Google Calendar

Architecture:
- cloud captures and triggers
- local runtime persists and processes
- Markdown is durable memory
- Notion/Google are projections
