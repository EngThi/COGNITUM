# SYSTEM RULES

1. Every important input becomes a CognitiveEvent.
2. Every important knowledge artifact eventually becomes Markdown.
3. Cloud tools are adapters, not the source of truth.
4. Critical logic lives in Python workers.
5. Do not introduce heavy infrastructure before daily usage is stable.
6. Prefer async I/O and low memory usage.
7. Never lose events.
8. Never expose secrets in logs.
